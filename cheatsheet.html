Ruby Enumerable Cheatsheet

Created by Nick Mascia and Alex Tarkowski

The Enumberable module provides a set of methods to traverse, search, sort and manipulate collections. Here is a brief cheatsheet of methods we find quite handy

Enumerable Explanation Example

EACH

 create an array. Each takes every element and perfoms the block on code on the elements. array = ["house", "man", "woman", "dog"]
array.each do |word|
puts word
end

=> house
=> man
=> woman
=> dog

EACH_WITH_INDEX

 Takes each element of the array and performs the block of code on the element and the index. numbers = ["zero","one","two"]
numbers.each_with_index do |number, index|
puts "#{index}: #{number}"
end

=> 0: zero
=> 1: one
=> 2: two

INJECT

 Reduces array of values to a single sum. If no number is explicitly given, inject will assume first number. numbers = [1,2,3].inject(0) do |total, sum|
total += sum
end
=> 6

STEP-BY-STEP OF WHAT INJECT DOES:
[1, 2, 3, 4].inject(0) { |result, element| result + element }
[2, 3, 4].inject(0 + 1) { |result, element| result + element }
[3, 4].inject((0 + 1) + 2) { |result, element| result + element }
[4].inject(((0 + 1) + 2) + 3) { |result, element| result + element }
[].inject((((0 + 1) + 2) + 3) + 4) { |result, element| result + element }
(((0 + 1) + 2) + 3) + 4
=> 10

SOURCE: http://stackoverflow.com/questions/710501/need-a-simple-explanation-of-the-inject-method

EACH_SLICE

 Performs block of code on a specified slice of a range or array [1,2,3,4,5,6].each_slice(2) { |slice| p slice }
=> [1,2]
=> [3,4]
=> [5,6]

ALL?

 Returns true if all elements satisfy the condition of the block CONDITION MET:
array = [1,2]
array.all? { |value| value == Integer }
=> true

CONDITION NOT MET:
array = [1,"dog", 2]
array.all? { |value| value == Integer }
=> false

ANY?

 Returns true if condition is met by any element in the array CONDITION MET:
array = [1,2]
array.any? { |value| value == Integer }
=> true

CONDITION NOT MET:
array = [1,2]
array.any? { |value| value == String }
=> false

GROUP_BY

 Will group elements according to if condition is satisfied as true. Will also group all false elements (1..10).group_by { |value| value.even? }
=> { false=> [1, 3, 5, 7, 9], true=>[2, 4, 6, 8, 10] }

TAKE_WHILE

 Will run all elements in array until the condition is no longer satisfied sentence = ["Hello", "My", "Name", "Is", "Nick", "Ok"]
sentence.take_while { |value| value != "Ok" }.join(" ")
=> "Hello My Name Is Nick"
