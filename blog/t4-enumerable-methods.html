Technical Blog 3: Enumerable#Group_by

Group_by is a very cool method in Ruby. It essentially allows an argument to be passed into an array, and then organized the information into groups.

Lets say we have an array that contains (dog, cat, mouse, lion, horse, fish). At this point the array is a list of objects with no organization. We can however break this down using group_by. For instance, say we want this list grouped by how long each word is. We would invoke grou_by.length on this list, and it would output some thing like this:

{[3=> dog, cat] , [4=>lion, fish], [5=>mouse, horse]}

As you can see, we have easily grouped the array by length. This would be quite useful in large lists for example, or arrays where you may want to group a list by first or last name.

We can also invoke group_by on hashes, and here is where we can really see the power of this method. It works like this:

group_by {|object| block} returns a hash.

When we use this on a hash, we have a greater ability for grouping and it unlocks being able to do so by keys and values. So for example if we have a list of people and also have their ages, we can group them according to either of these. Its pretty powerful stuff when you consider how simply it returns the desired result.
