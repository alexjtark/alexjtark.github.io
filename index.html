Responsive Design. Always.

According to SimilarWeb's State of Mobile Web US 2015 report, roughly 56 percent of consumer traffic to the leading US websites is now from mobile devices. You need to work with a developer that understands responsive web design so that you are not losing visitors/potential sales due to poor mobile viewing experience.

I understand the core philosophy of building responsive sites, and always build with a mobile first approach.
